# Mini Third Person Game (Tkinter)

Jogo simples em Python simulando visão em terceira pessoa com movimentação WASD, câmera seguindo o player e inimigos com IA básica.

## Conceito
Player se move em um mundo 2D com paredes, câmera acompanha suavemente, inimigos perseguem usando vetor direção normalizado.

## 🎮 Controles
- WASD → mover
- R → resetar jogo
- ESC → sair

## Requisitos
- Python 3
- Tkinter (já incluso na maioria das distros)

## Como rodar
python main.py
